# Installation

## Npm Install

```bash
$ npm install --save @naraejs/webserver-express
```

## Basic Code

```typescript
import * as naraejs from '@naraejs/core';
import * as webserverExpress from '@naraejs/webserver-express';

naraejs.install(webserverExpress);
```
